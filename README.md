# Calculator Project

## Description
This project is a simple calculator application designed to perform basic arithmetic operations.  
It allows users to input values and calculate results efficiently.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly input and output

## Technologies Used
- Bash Shell Script
- Git and GitHub

## How It Works
The calculator accepts user input, processes the selected arithmetic operation, and displays the computed result.
